# Clicker-Game-Cmd

This is a command prompt clicker game!

## Controls

| Key | Action |
|-----|--------|
| <span style="color:red">**1**</span> | Click to earn points |
| <span style="color:red">**2**</span> | Upgrade your clicker |
| <span style="color:red">**3**</span> | Admin screen |
| <span style="color:red">**X**</span> | Close game |

## About

This is a new clicker game built for the command prompt. Click your way to victory!
